# Contextual Vencord Plugin

## Development Setup

This plugin is designed to be loaded into [Vencord](https://github.com/Vendicated/Vencord).

1. Clone Vencord: `git clone https://github.com/Vendicated/Vencord`
2. Create a symlink or copy this folder into `Vencord/src/userplugins/contextual`
3. Build Vencord: `pnpm build`
4. Inject Vencord: `pnpm inject`

## Structure
- `index.tsx`: Main plugin entry point
